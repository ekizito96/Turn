# Turn Language Support

This extension provides syntax highlighting and Language Server features (Hover, Go-To-Definition, Diagnostics) for the Turn agentic systems language.

## Features

- Syntax highlighting for `.tn` files.
- Integrates with the `turn lsp` binary for deep static analysis.

## Requirements

Ensure the `turn` compiler is installed and available in your environment PATH, or run this extension within the Turn compiler repository directly.
